The **{$frontmatter.title}** action calls given function when mouse leaves the page

```svelte|copy|lineNumbers
<script>
    import { pageleave } from '@svelteuidev/actions';

    $: count = 0;
</script>

<div use:pageleave={() => count++}>{count}</div>
```
